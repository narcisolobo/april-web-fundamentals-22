Position Property

Rule #1: Do Not ABUSE POSITION!!

Very rarely will you ever have to use position.

If an element is "breaking the plane" of another element, that's a good time to use position.

The Values of the Position Property:

All values are relative - it's just a question of what they are relative to.

Static

The default value of position. Top, right, bottom, left has no effect.

Relative

The element is positioned relative to its normal position. Top, right, bottom, left have effect.

Absolute

The element is positioned relative to its closest positioned ancestor. Top, right, bottom, left have effect.

An absolutely positioned element should have a relatively positioned parent.

Fixed

The element is positioned relative to the browser window. Top, right, bottom, left have effect.

This value is useful for cookie warnings, for example.

The element toggles between relative and fixed. When the element reaches the user-defined position, it becomes fixed.

This value is useful for headers that remain at the top of the page as you scroll.
